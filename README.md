AI & Machine Learning Practice Projects

This repository contains my hands-on practice projects from AI-ML sessions.
The goal is to understand data analysis, visualization, and basic machine learning models.
 
 Project 1: Loan Approval Prediction
 Objective

To predict whether a loan will be approved based on applicant details.

--Dataset Description--
Includes features like:
Applicant income
Loan amount
Credit history
Dependency status

--Technologies Used--
Python
Pandas
NumPy
Matplotlib / Seaborn

--Key Tasks Performed--
Handling missing values
Encoding categorical variables
Model training and testing
Accuracy evaluation

--Results--
Built a classification model to predict loan approval
Achieved good accuracy on test data

--Learning Outcomes--
Understood real-world datasets
Improved data visualization skills
Learned basic machine learning workflow
Gained hands-on experience with model building

Project 2: Student Performance Analysis

--Dataset Description--

Study hours
Attendance
Previous scores
Extracurricular activities
Final performance

--Technologies Used--
Python
Pandas
NumPy
Matplotlib
Seaborn

--Key Tasks Performed--
Data cleaning and preprocessing
Data visualization using graphs
Correlation analysis
Train-Test Split 
Model Building

--Key Insights--
Students with higher study hours tend to perform better
Attendance has a strong impact on performance
Previous scores influence final results
